# ADX Engine Design System

This folder contains brand and design specifications for use across all ADX Engine projects —
including client deliverables, dashboards, and operator interfaces built outside this repo.

## Files

| File | Purpose |
|---|---|
| `BRAND.md` | Complete brand spec for external AI agents — colors, typography, components, voice, layout |

## How to Use

When starting a new project or briefing an external agent:

1. Open `BRAND.md`
2. Copy the full contents
3. Paste as a system instruction or initial context to the agent
4. The agent now has everything needed to produce on-brand output without re-briefing

## Source of Truth

The adxengine.net codebase (`/src`) is the implementation reference.
This folder is the specification reference — written for agents, not browsers.
